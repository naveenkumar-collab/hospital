# **App Name**: MediTrack

## Core Features:

- Centralized Patient Records: Securely store and manage patient information, including medical history, appointments, and billing details.
- Appointment Scheduling: Allow patients to book appointments online and enable staff to manage doctor schedules efficiently.
- Automated Billing System: Generate and manage bills automatically, reducing errors and long waiting times.
- Staff Management: Manage staff information, including roles, schedules, and contact details.
- Smart Task Routing: LLM Tool routes notifications to departments or workers to ensure better intra-office communications and minimize the need for inter-office emails.
- Report Generation: Generate comprehensive reports on hospital operations, patient statistics, and financial performance.
- Database: Use a cloud SQL database to save user data.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to convey trust, cleanliness, and health.
- Background color: Very light blue (#F0F8FF), close to white, for a clean, sterile feel.
- Accent color: A muted teal (#70A1AF), approximately 30 degrees away from blue, for secondary interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, objective look.
- Use clear, easily recognizable icons for different functionalities, such as calendar icons for appointments and document icons for patient records.
- Clean and intuitive layout with clear sections for different functionalities to improve user experience.
- Subtle animations for loading screens and transitions to enhance user engagement without being distracting.
